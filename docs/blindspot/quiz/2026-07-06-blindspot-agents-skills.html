Pre-Merge Quiz: blindspot Agent/Skill 저장소 구축

변경 요약

  • 빈 저장소를 여러 프로젝트가 함께 쓰는 AI 작업 도구 모음으로 만들었습니다.
  • 작업 절차 도구(스킬) 5종: 요구사항 인터뷰, 사각지대 점검, 설명 문서 작성, 작업 보고, 그리고 이 네 단계를 순서대로 진행해 주는 진행 도구.
  • 보조 조사원(에이전트) 3종: 코드 조사, 문서 검증, 변경 분석 — 모두 읽기만 하고 고치지는 않습니다.
  • 규칙 강제: 작업 세션이 시작될 때마다 '작업 유형별 필수 절차' 규칙 문서를 AI에게 자동으로 읽혀 주고, 프로젝트 안내문에서도 같은 문서를 참조하는 이중 안전망.
  • 설치: 설치 프로그램 한 번 실행으로 프로젝트에 연결되며, 여러 번 실행해도 결과가 같습니다(멱등성).
  • 자동 점검: 규칙 알림 동작, 문서 형식, 그리고 가짜 프로젝트에 두 번 설치해 보는 반복 설치 검사까지 포함.
  • 마지막 검토 반영: 깨진 설정 파일을 만나면 즉시 멈추고 알리기, 코드가 없는 새 프로젝트의 처리 방식 문서화 등.
