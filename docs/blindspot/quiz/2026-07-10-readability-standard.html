Pre-Merge Quiz: 산출물 가독성 표준 도입

변경 요약

  • AI가 만드는 다섯 가지 작업 문서(요구사항, 사각지대 점검, 설계 설명서, 작업 보고서, 확인 퀴즈)에 쉬운 문장 기준이 생겼습니다.
  • 질문·결정·요약·해설은 쉬운 말로 씁니다. 코드 검토용 안내와 AI용 기록은 기술 표현을 그대로 둡니다.
  • 확인 퀴즈에 문항별 해설이 생겼습니다. 정답 확인을 누르면 해설이 나타나고 정답 보기가 강조됩니다. 예전 퀴즈는 맞고 틀림만 알려 줬습니다.
  • 해설은 필수 규칙이지만, 화면 코드에 누락을 걸러 주는 장치는 없습니다. 해설을 빠뜨리면 그 자리에 'undefined'라는 영문 글자가 그대로 보입니다.
  • 문장 규칙을 검사하는 자동 테스트는 없습니다. 대신 문서 검사 보조 AI와 '저장 전 자가 점검' 지침이 규칙을 지킵니다.
  • 배포 직전 마지막 검토에서 빈틈 하나를 고쳤습니다. 미해결 질문 표의 '질문' 칸이 쉬운 문장 규칙의 적용 목록에서 빠져 있었습니다.
  • 예전에 만든 퀴즈와 보고서 요약도 새 기준으로 다시 썼습니다. 통과 기록은 그대로 보존했습니다.
  • 이 도구를 쓰는 다른 프로젝트는 재설치가 필요 없습니다. 연결이 바로가기(심링크) 방식이라 저장소만 최신으로 받으면 반영됩니다.
  • 사용 안내서(README)와 개발 지침(CLAUDE.md)도 함께 갱신했습니다.
