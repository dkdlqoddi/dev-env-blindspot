Pre-Merge Quiz: 산출물 가독성 표준 도입

변경 요약

  • AI가 만드는 다섯 가지 작업 문서(요구사항, 사각지대 점검, 설계 설명서, 작업 보고서, 확인 퀴즈)가 '코드를 본 적 없는 사람도 읽을 수 있는 문장' 기준을 갖게 됐습니다.
  • 같은 문서 안에서도 읽는 사람이 다른 부분은 구분합니다 — 질문·결정·요약·해설은 쉬운 말로, 코드 검토용 안내·AI용 기록·조사 증거는 기술 표현 그대로.
  • 확인 퀴즈에 문항별 해설이 생겼습니다 — 정답 확인을 누르면 해설이 나타나고 정답 보기가 강조됩니다.
  • 예전에 만든 퀴즈와 보고서 요약도 새 기준으로 다시 썼습니다 (통과 기록은 그대로 보존).
  • 사용 안내서(README)와 개발 지침(CLAUDE.md)도 새 기준에 맞춰 함께 갱신했습니다.
